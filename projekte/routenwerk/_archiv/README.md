# _archiv

Ablage fuer bereits fertig entwickelte Assets (Icons, SVGs, Design-Snippets) aus diesem
Projekt, damit sie bei Bedarf wiederverwendet/reproduziert werden koennen, statt sie neu
zu designen. Kein dauerhaftes Fundus-Archiv (siehe dafuer _fundus/ im Workspace-Root) —
wird von Marvin bei Gelegenheit manuell bereinigt.
